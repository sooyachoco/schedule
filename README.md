# Jiyeon's Schedule

모바일 최적화 일정 관리 웹앱

## 기능

- **일간 뷰** — 시간(30분 단위) + 일정 입력, 대표일정 2개 체크
- **주간 뷰** — 대표일정 리스트 카드, 날짜 탭 시 일간으로 이동
- **월간 뷰** — 캘린더 그리드에 대표일정 표시
- **날짜 선택** — 드롭다운 미니 캘린더로 빠른 이동
- **스와이프** — 좌우 스와이프로 이전/다음 탐색
- **localStorage** — 브라우저에 데이터 자동 저장
